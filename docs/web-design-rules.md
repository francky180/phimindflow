# WEB DESIGN RULES

Design this website like a world-class luxury conversion funnel.

Visual direction:
- premium light theme (off-white #FAFAFA)
- white cards with subtle shadows
- near-black text (#0A0A0A)
- restrained gold accents (#C9A84E)
- elegant typography
- spacious layout with breathing room
- strong alignment
- clean grid structure
- confident section spacing (py-28+)

CTA order:
1. Open Broker Account
2. Get Course Access
3. Continue to Management

Avoid:
- generic startup look
- cheap gradients
- clutter
- weak spacing
- random design decisions
- over-animation
